# Эфир: Кто будет расти в 2026 году

## What This Is

Лендинг + Telegram-бот для регистрации на бесплатный эфир Вадима Сорокина (EGZ Academy) 24 марта 2026, 19:00 МСК. Целевая аудитория — тёплая база маркетологов из Telegram. Цепочка: лендинг → Telegram-бот (регистрация) → Google Таблица (база) → рассылки-напоминания.

## Core Value

Человек с лендинга за один клик попадает в Telegram-бот, регистрируется на эфир, данные автоматически попадают в Google Таблицу для аналитики и рассылок.

## Context

**Продукт:** Бесплатный эфир для маркетологов о системе заработка 500к–1 млн в маркетинге.

**Спикер:** Вадим Сорокин, владелец маркетингового агентства EGZ.

**Аудитория:** Тёплая — знают Вадима, видели прогрев в Telegram. Преимущественно мобильный трафик.

**Дата эфира:** 24 марта 2026, вторник, 19:00 МСК.

**Структура лендинга (4 блока):**

1. **Первый экран (hero):** Логотип EGZ Academy, заголовок «Кто будет расти в 2026 году», подзаголовок про разницу в заработке маркетологов, мета (дата/время/бесплатно), фото Вадима, подпись, CTA «Зарегистрироваться»
2. **Что разберём:** 5 пунктов программы (позиционирование, лидген, продажи, удержание клиентов, CRM/воронка), CTA
3. **Кому стоит прийти:** 5 болей ЦА, CTA
4. **Финальный блок:** Обратный отсчёт до 24 марта 19:00 МСК, основная CTA → бот, вторичная CTA «Задать вопрос» → ЛС Вадима

**Telegram-бот:** SaleBot уже подключен как сервис. Нужно реализовать бота (самописный или через SaleBot API): deep link из лендинга, приветственное сообщение, кнопка регистрации, подтверждение, запись в Google Таблицу. Рассылки: за день, за час, ссылка на эфир, после эфира — запись + CTA.

**Дизайн-референсы:** Тёмный фон, светлый текст, крупный шрифт, минимализм. Приложены 3 скриншота-референса (Vortek VR, Terrixa, FRZN — тёмные футуристичные лендинги).

**Фото Вадима:** Файл есть, будет подставлен.

**Ссылка на бота:** Пока нет, используем placeholder.

## Constraints

- **Деплой**: GitHub Pages — только статика (HTML/CSS/JS), без серверного бэкенда
- **Бот**: SaleBot как платформа, либо самописный бот
- **Дизайн**: Обязательно использовать навык frontend-design для UI/UX
- **Mobile-first**: Аудитория из Telegram = мобильные устройства
- **Срок**: Эфир 24 марта 2026 — лендинг и бот должны быть готовы заранее
