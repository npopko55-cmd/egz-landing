## Spacing Scale

Declared values (multiples of 4 only):

| Token | Value | CSS Variable | Usage |
|-------|-------|-------------|-------|
| xs | 4px | `--space-xs` | Icon gaps, badge padding, tight inline elements |
| sm | 8px | `--space-sm` | Compact element spacing, label-to-input gaps |
| md | 16px | `--space-md` | Default padding inside cards and buttons |
| lg | 24px | `--space-lg` | Section internal padding, list item gaps |
| xl | 32px | `--space-xl` | Block-to-block spacing within a section |
| 2xl | 48px | `--space-2xl` | Section break between the 4 landing blocks |
| 3xl | 64px | `--space-3xl` | Page-level top/bottom padding (hero, final block) |

Exceptions:
- Touch targets for CTA buttons: minimum 48px height (mobile accessibility, WCAG 2.5.5)
- Sticky CTA bar bottom-safe-area: `padding-bottom: env(safe-area-inset-bottom, 16px)` — iOS home indicator clearance
- Countdown timer card cells: 60px wide × 72px tall (mobile), 80px × 96px (desktop) — sized for legibility of 2-digit numbers

| Role | CSS Variable | Size | Weight | Line Height | Usage |
|------|-------------|------|--------|-------------|-------|

| Heading | `--text-heading` | 20px | 600 | 1.3 | Block section headers, card titles, program item headings |

- Only 2 weights used across all text: 400 (regular) and 600/700 (semibold/bold). 700 reserved for display headline only.
- Letter-spacing for display: `-0.02em` (tighten large headlines per futuristic premium style).
- All countdown digits: 48px mobile / 64px desktop, weight 700, font-variant-numeric: tabular-nums.

Dark futuristic theme. References: Vortek VR, Terrixa, FRZN. Source: CONTEXT.md + REQUIREMENTS.md (LAND-01, LAND-03).

| Role | CSS Variable | Value | Usage |
|------|-------------|-------|-------|
| Dominant (60%) | `--color-bg` | `#0A0A0F` | Page background, section fills |
| Secondary (30%) | `--color-surface` | `#12121A` | Cards, countdown digit containers, pain-point list items, program block rows |
| Accent (10%) | `--color-accent` | `#7C3AED` | Reserved list — see below |
| Accent glow | `--color-accent-glow` | `rgba(124, 58, 237, 0.35)` | box-shadow glow on accent elements |
| Accent gradient start | `--color-accent-alt` | `#A855F7` | Gradient terminus on CTA buttons |
| Text primary | `--color-text` | `#F0F0F8` | All body and heading text |
| Text muted | `--color-text-muted` | `#8888A8` | Meta labels, countdown unit labels, secondary copy |
| Border subtle | `--color-border` | `rgba(255,255,255,0.08)` | Card borders, dividers, countdown cell borders |
| Destructive | `--color-destructive` | `#EF4444` | Not applicable in Phase 1 — no destructive actions |

Accent (`#7C3AED`) reserved exclusively for:
1. Primary CTA button background gradient (`#7C3AED` → `#A855F7`)
2. Primary CTA button hover glow (`box-shadow: 0 0 24px var(--color-accent-glow)`)
3. Sticky CTA button (same gradient as primary CTA)
4. Active/focus ring on interactive elements (outline: 2px solid `var(--color-accent)`)
5. Countdown timer digit background subtle left-border accent line (`border-left: 2px solid var(--color-accent)`)
6. Section heading underline accent stripe (4px × 40px block under section `<h2>` titles)

Accent is NOT applied to: links (use text-primary), borders (use color-border), secondary CTA (use ghost style), general text.

Background texture: subtle radial gradient noise on `--color-bg`:
```
background: radial-gradient(ellipse at 20% 50%, rgba(124,58,237,0.08) 0%, transparent 60%),
            radial-gradient(ellipse at 80% 20%, rgba(168,85,247,0.06) 0%, transparent 50%),
            #0A0A0F;
```

## Interaction Contract

### Scroll Animations (LAND-04)
- Mechanism: `IntersectionObserver` with `threshold: 0.15`
- Initial state: `opacity: 0; transform: translateY(24px)`
- Triggered state: `opacity: 1; transform: translateY(0)`
- Transition: `opacity 0.5s ease, transform 0.5s ease`
- Stagger: successive children delay by `0.1s` increments via inline `transition-delay`
- Respect `prefers-reduced-motion`: if set, skip transform, only fade (or skip entirely)

### Sticky CTA Visibility (LAND-10)
- Show sticky CTA when hero section's primary CTA button scrolls out of viewport
- Mechanism: `IntersectionObserver` on hero CTA button (`rootMargin: '0px'`, `threshold: 0`)
- When `isIntersecting: false` → add class `sticky-cta--visible` → `translateY(0)`
- When `isIntersecting: true` → remove class → `translateY(100%)`
- Transition: `transform 0.3s ease`

### Non-hero CTA Buttons (LAND-12)
- Hero, mid-section, and pain-points CTA buttons: smooth scroll to final block
- `document.querySelector('#final-block').scrollIntoView({ behavior: 'smooth' })`
- Final block CTA: navigates to deep link `https://t.me/<botname>?start=landing` (opens Telegram)

### Countdown Timer (LAND-11)
- Target: `2026-03-24T19:00:00+03:00` (explicit offset, not timezone name)
- Update interval: `setInterval` every 1000ms
- Display: DD days, HH hours, MM minutes, SS seconds
- Unit labels (Russian): «ДНИ», «ЧАС», «МИН», «СЕК» — always visible in `--color-text-muted`
- Label «МСК» displayed as static tag adjacent to the timer block (not inside cells)
- Post-deadline: when `Date.now() >= target`, replace countdown row with text «Эфир уже идёт!» in `--color-accent`, CTA button remains

### Photo Placeholder (LAND-16)
- `<img src="assets/vadim-photo.jpg" alt="Вадим Сорокин">` with placeholder fallback
- Avatar wrapper applies radial glow ring via CSS `::after` pseudo-element
- If image fails to load (`onerror`): show initials block «ВС» styled as avatar
